# Zenvest Frontend

Public-facing landing page and authentication portal for the Zenvest investment platform built with React and Vite.

## Features

- Responsive landing page
- User authentication (Login/Signup)
- Email verification
- Public information pages (Home, About, Products, Pricing, Support)

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Technologies

- React 18
- Vite
- ESLint
- CSS3

## Project Structure

- `src/` - Source code
- `src/landing_page/` - Landing page components
- `src/context/` - React context for authentication
- `public/` - Static assets
